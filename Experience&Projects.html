Project Name:Simple Payment Wallet
    Highlights of the project:
  • Basic User Registration and Authentication
  • Users can register an account.
  • Secure login functionality (basic session or token-based on localhost).
  • Add Funds / Load Wallet
  • Users can add money to their wallet balance.
  • Simple fund addition interface with input validation.
  • Send / Transfer Money
  • Users can transfer funds to other registered users.
  • Wallet balance updates accordingly.
  • Transaction History
  • Each user can view their previous transactions (sent, received, loaded).
  • Display of transaction date, amount, and recipient details.
